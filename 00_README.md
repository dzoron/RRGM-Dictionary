# RRGM Dictionary v1.0 Breakout

This package splits the RRGM Dictionary into one file per letter and grouped reading bundles.

## Contents

- `by_letter/`: A-Z source-preservation letter files.
- `grouped_for_reading/`: larger bundles for human reading or corpus loading.
- `source_inputs/`: copies of the working source files used to produce this breakout.
- `corpus_notes/`: weight/use notes for the Beach Luna Paradox process trace.
- `PROVENANCE.md`: source and repair notes.
- `REPAIR_LOG.md`: explicit recovery and cleanup actions.
- `VALIDATION_REPORT.md`: numbering and count validation.
- `manifest.json`: machine-readable counts and warnings.
